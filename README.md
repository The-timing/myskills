# 个人技能库

这是用于 CC Switch / Claude 的个人经验技能库。根目录仅保留会自动判断、可直接导入的技能；`_archive` 是原始经验备份，不参与日常导入。

在 CC Switch 中选择“导入已有”，按需求导入下面某个具体目录，不需要 ZIP。

## 主技能

| 技能 | 适用场景 |
| --- | --- |
| `lanyan-delivery-guardrails` | 多端联动、线上问题、截图/curl/报错驱动的完整交付 |
| `lanyan-backend-data` | Spring Boot、MyBatis-Plus、SQL、接口、回调和状态流转 |
| `lanyan-admin-console` | Vue2 ElementUI 后台列表、筛选、详情、权限和菜单 |
| `lanyan-miniapp-client` | uni-app/微信小程序请求、图片、分享、输入、提示和页面状态 |
| `lanyan-operation-content` | 运营配置、协议、客服、积分规则、富文本和 UEditor |
| `lanyan-commerce-orders` | 商品/SKU/分类、券、积分兑换、订单和支付成功 |
| `lanyan-team-operations` | 团队、成员、奖励、业绩、公告和定时结算 |
| `lanyan-third-party-media` | 三方签名/回调、微信支付、上传、图片和分享海报 |
| `lanyan-experience-intake` | 把新问题沉淀为不重复、可验证的经验 |

## 维护原则

- 新经验优先补充到一个已有主技能，不按每次问题新建技能。
- 每条经验只保留“触发、根因、约束、验证”四件事。
- 公共第三方技能保持原样；个人历史版本和原始资料统一放在 `_archive`。
