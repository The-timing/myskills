# 蓝湖切图优先通过图片信息 JSON 定位导出资源

## 1. 问题描述

- 用户往往不是要整张设计稿，而是要蓝湖页面内部某个具体图层的导出资源，例如：
  - 文章封面图
  - 按钮图
  - 图标
  - 卡片配图
- 在人工操作时，通常可以通过“点击元素 -> 右侧切图面板 -> 下载切图”完成。
- 但在自动化环境下，常见问题有：
  - 右侧切图面板不稳定展开
  - 标注详情页 `detailDetach` 变成空白
  - 只能看到整张稿图，难以精确映射到某个图层

## 2. 触发条件与影响范围

- 触发条件：
  - 用户要求拿蓝湖原生切图，而不是本地裁图。
  - 已经可以登录蓝湖并进入项目。
  - 能通过页面名定位到目标设计页面。
- 影响范围：
  - 设计资源提取流程
  - 自动化浏览器脚本
  - 批量素材整理效率

## 3. 复现步骤

1. 打开蓝湖项目页，找到目标页面，例如“护肤百科”。
2. 尝试直接进入 `detailDetach` 或依赖右侧切图面板提取某个图层。
3. 在自动化环境下观察：详情页可能空白，或只弹出悬浮操作条，未稳定显示完整下载面板。
4. 如果此时改走蓝湖“图片信息”中的 JSON 与导出资源地址，则仍可稳定拿到目标图层切图。

## 4. 根因分析

- 根因结论：
  - 蓝湖右侧切图面板只是一个前端交互入口，真正稳定的资源来源是“图片信息”里的结构化 JSON 和导出资源地址。
- 直接原因：
  - 自动化环境下，面板展开、悬浮态和详情页布局计算都可能不稳定。
  - `detailDetach` 页面中承载主图的容器有时会被算成 `0x0`，导致页面看起来空白。
- 深层原因：
  - 处理流程过度依赖前端可视面板，没有回到蓝湖内部的结构化导出数据。
- 为什么此前没有暴露：
  - 人工点击时，面板通常能正常展开，问题更多出现在自动化脚本和远程浏览器环境中。

## 5. 分步处置方案

1. 先通过页面列表锁定目标页面

```text
请求：
/api/project/images?project_id=<pid>&team_id=<tid>&dds_status=1&position=1&show_cb_src=1&comment=1

目标：
- 按页面名定位目标页面
- 拿到 image_id、name、url
```

注意事项：

- 优先按页面名定位，不要直接在整块画布上盲点。
- `image_id` 是后续进入目标画布页和内部对象定位的关键字段。

2. 回到 stage 画布页，并调用蓝湖自身“图片信息”

```text
路由：
#/item/project/stage?pid=<pid>&image_id=<image_id>&tid=<tid>

组件入口：
document.querySelector('.project_stage_view').__vue__

调用方式：
vm.doShowImageInfo(item)
```

注意事项：

- `item` 可从 `vm.stage.imagesIdMap[image_id]` 中获取。
- 不要把 `detailDetach` 空白误判成“无数据”，切图提取优先在 `stage` 完成。

3. 解析图片信息中的图层和导出资源

```text
重点字段：
- artboard.layers
- frame / combinedFrame
- hasExportImage / hasExportDDSImage
- assets
```

推荐映射方式：

- 先按图层坐标定位目标区域，例如：
  - 第一个文章封面位于左侧列表区
  - 尺寸大约为 152 × 86
  - top 落在首个文章卡片区域
- 再把命中的图层与 `assets` 中的 `FigmaSlicePNG` / `FigmaSliceSVG` 对应起来。

注意事项：

- 优先返回蓝湖导出的切图 URL，不要退回本地裁图。
- 如果页面中有多个同名图层，必须结合位置坐标做二次确认。

## 6. 验证与验收标准

- 功能验证点：
  - 能通过页面名稳定定位目标页面。
  - 能拉起“图片信息”并解析出 `artboard.layers` 和 `assets`。
  - 能返回目标图层对应的 `FigmaSlicePNG` 或 `FigmaSliceSVG` 资源地址。
- 回归验证点：
  - 即使右侧切图面板未稳定出现，仍可通过图片信息完成提取。
  - 即使 `detailDetach` 页面空白，仍可在 `stage` 提取切图。
- 可执行测试用例：
  - 以“护肤百科”页面为例，定位第一个文章封面图层。
  - 解析图片信息中的首个封面图层位置。
  - 打开对应 `FigmaSlicePNG` 地址，确认内容与页面内第一篇文章封面一致。
- 通过判定条件：
  - 返回的是蓝湖导出的图层切图 URL，而不是整张设计稿裁图结果。
